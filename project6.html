This is my sixth page of my website!!
This is small PHP Chat Application, developed using PHP and HTML and deployed using Herokuapp. I have recently learnt about PHP and its backend uses. So I haven't made anything bigger.

In this PHP Chat Application users can join the chat session by entering their name on the main page. User will be redirected to the chat session where other users will be present in the session. Users can chat with one another and multiple users can join the session. For now users can send only text messages, in the future updates users will be able to send images and videos.

The purpose of the website is to learn about how PHP can be integrated inside HTML and develop cool things. In this project no database is used to store the messages, so the messages disapper once the server goes in stand by mode.
