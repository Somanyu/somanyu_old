Basic Banking System

A Short Project

This is my fourth page of my website!!
This website is basically about creating a basic banking system. I developed this with the help of PHP, HTML, MariaDB and bit of CSS and Bootstrap both for styling the website.

I started by creating the first page of the website. I kept it simple i.e. no sign up, login, etc. Just a simple page with button which redirects to another site where you will be able to see all the customers with their email ids', balance, their name and with option for transfering amount to other customers. I used MariaDB to store customers details. On the transfer page user can transfer amount available to them. After a successfull transaction you can see the history of transaction done by various users.

Coming to the design of the website, I tried to keep as simple as that. I chose red as primary colour and white as secondary colour. I used bootstrap for creating tables and transaction page. There isn't much deisgn but it's simple and basic. More updates will come to this banking system.
